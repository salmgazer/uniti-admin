# Uniti Admin Panel

Admin panel for the Uniti application.

## Features

- User management
- Goal management
- App management
- Admin user management
- Dashboard with statistics

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Default Admin Credentials

- Email: admin@uniti.com
- Password: admin123

## API Integration

The admin panel connects to the Uniti API running on http://localhost:3000. Make sure the API is running before using the admin panel.
